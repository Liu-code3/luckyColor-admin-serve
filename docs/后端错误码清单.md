# LuckyColor Admin Serve 后端错误码清单

## 1. 说明

本文档用于统一后端业务错误返回规范。

- 成功返回固定为：`{ code: 200, msg: "success", data }`
- 失败返回固定为：`{ code, msg, data: null }`
- HTTP 通用错误继续保留标准状态码，例如 `400/401/403/404/422/500`
- 业务错误码使用独立编号，便于前后端联调、日志检索、权限与租户场景区分

## 2. 编码规则

- `1011xxx`：认证中心
- `1012xxx`：权限中心
- `1013xxx`：租户中心
- `1014xxx`：系统管理主数据
- `1015xxx`：文件服务
- `1016xxx`：通用业务校验

## 3. 错误码表

### 3.1 认证中心

- `1011001` 用户名或密码错误
- `1011002` 当前账号已被禁用
- `1011007` 登录已过期，请重新登录
- `1011008` 登录状态无效，请重新登录

### 3.2 权限中心

- `1012001` 当前账号没有此操作权限
- `1012002` 当前账号没有此菜单权限
- `1012003` 当前账号没有此数据权限

### 3.3 租户中心

- `1013001` 租户不存在
- `1013002` 当前租户已被禁用
- `1013003` 当前租户已过期
- `1013004` 当前账号不能访问该租户
- `1013007` 当前租户已被冻结

### 3.4 系统管理主数据

- `1014001` 用户不存在
- `1014002` 角色不存在
- `1014003` 菜单不存在
- `1014004` 部门不存在
- `1014005` 字典不存在
- `1014006` 系统配置不存在
- `1014007` 通知公告不存在

### 3.5 文件服务

- `1015001` 文件不存在
- `1015002` 文件上传失败
- `1015003` 文件删除失败

### 3.6 通用业务校验

- `1016001` 请求参数校验失败
- `1016002` 数据已存在，请勿重复创建
- `1016003` 当前状态不允许执行该操作

## 4. 当前落地规则

- 认证失败统一返回 `1011001`
- 登录态失效统一返回 `1011008`
- 用户不存在统一返回 `1014001`
- 菜单不存在统一返回 `1014003`
- 字典不存在统一返回 `1014005`
- 文件不存在统一返回 `1015001`
- DTO 参数校验失败统一返回 `1016001`
- 其他未接入业务错误码的异常，继续按 HTTP 状态码返回

## 5. 后续接入建议

- 角色、部门、系统配置、通知公告模块接入后，统一替换为 `1014002/1014004/1014006/1014007`
- 权限校验失败，统一替换为 `1012001`
- 租户禁用、租户过期、租户冻结接入后，统一使用 `1013002/1013003/1013007`
- 文件上传与删除失败场景，进一步细化接入 `1015002/1015003`
- 唯一性校验与状态流转场景，逐步接入 `1016002/1016003`
